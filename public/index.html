React Dynamic Form Generator

Given a form definition, provided as a JSON array, generates the form as HTML. Each item in the array is an object with these required properties:

tag
HTML tag to use (e.g., "input")
name
Internal identifier for the form field
type
HTML tag type to use (e.g., "text" or "checkbox")
human_label
Human-readable identifier for the form field

When you press submit, the form data will be logged to the console.
